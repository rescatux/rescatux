Help Rescapp - Documentation This is Rescapp. The main Rescatux UI.

We are going to make a tour through its options. The first thing you should know is that you can always go back to the first Rescapp screen thanks to the MAIN MENU option.

If you happen to need help is good to know that at the upper left part of the UI the Support options are located.

  • Chat: Open Xchat window so that can you ask help on the Rescatux IRC channel.
  • Share log: Generates a pastebin from an existing log file so that can ask help in the chat window with your data at hand.
  • Help: Opens the help you are currently reading just in case you're not sure of something

Below the main menu and support options you will find the group options. That is options that have other options inside.

You might choose the Grub(+) option which will lead you to Grub options.

Once there you can see Restore Grub option. Clicking on it will show its information on the bottom web-browser alike tab but it will not run it.

If you want to run it you will have to click on the name of option plus !!! option. In our example you will have to click on Restore Grub !!!.

While navigating through the different options the Back option will appear so that you can go back to previous menu.

You have to know that apart from the options found thanks to the Rescapp UI there are other tools announced in Rescatux. These tools will probably be found thanks to the Start menu. These tools are not supported by Rescatux team so they are supposed to be used by experts even if they are easy to use.

Finally we will learn how to shutdown Rescatux. The powerdown button is found at the right bottom corner.(I inside O).

You will then be prompted what you want to do. Either you will select Shutdown to turn off your computer or reboot to use it inmediately.

Finally you are asked to remove your media (cdrom or usb) from your computer and press a key to continue so that the computer can be shutdown or rebooted.
